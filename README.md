🔢 Fibonacci Series Program (C++)

This project contains a simple C++ program to generate the Fibonacci series up to n terms.

---

📌 What is Fibonacci Series?

The Fibonacci series is a sequence where each number is the sum of the two preceding ones.

Example:
0, 1, 1, 2, 3, 5, 8, 13 ...

---

💻 Code Description

- Uses basic C concepts
- Takes input from user
- Prints Fibonacci series up to given number of terms

---

▶️ How to Run

1. Open the program in any C++ compiler
2. Compile the code
3. Run the program
4. Enter number of terms

---

🧾 Example

Input:
Enter number of terms: 5

Output:
0 1 1 2 3

---

⚠️ Note

This program is written using Turbo C++ style:

- Uses "conio.h"
- Uses "clrscr()" and "getch()"

These may not work in modern compilers.
